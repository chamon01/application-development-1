# Domain: Task / Todo System

This API supports a task management system where authenticated users create lists and tasks, update task state (like completed), and retrieve tasks with filtering and pagination. The contract is resource-oriented (nouns), uses consistent HTTP semantics, and is designed to evolve without breaking clients.

## Who uses the API
A web or mobile client used by authenticated end users to manage their own tasks and lists.

## Resources
- users
- lists
- tasks
